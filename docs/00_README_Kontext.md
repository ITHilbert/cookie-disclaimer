# Kontext & Zielsetzung: ITHilbert Cookie Disclaimer

Dieses Paket stellt Funktionalitäten für das Cookie-Consent-Management bereit.

## Ziel
Rechtssichere und konfigurierbare Verwaltung von Cookies und Tracking-Skripten.

## Hauptfunktionen
- Anzeige des Cookie-Banners
- Verwaltung von Zustimmung (Consent)
- Blockieren von Skripten vor Zustimmung
